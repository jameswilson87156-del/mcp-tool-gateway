# Design References

This folder stores AI-generated concept references for MCP Tool Gateway.

The concept images are not real product screenshots and must not be used in the main README as running application evidence. They define the visual direction only: light Linear / Stripe style developer console, Chinese-first enterprise SaaS copy, graphite text, cool gray borders, cobalt actions, green success, orange review, and red blocked states.

Frontend implementation must translate these references into real DOM, CSS, and Vue components. Runtime screenshots for README must come from Playwright captures under `docs/images`.
